日本語WordNetをJavaから利用するためのライブラリです． プロジェクトの詳細及び使用方法は，SourceFoege.JP内のプロジェクトページを参照して下さい．

必須環境

  • Java SE 6 以降
  • 日本語WordNetデータファイル
  • SQLiteJDBCドライバ (オプショナル)

サンプルコード

// データファイルを指定してWnj2のインスタンスを作成します．
Wnj2 wn = new Wnj2(new File("./wnjpn.db"));

// Synsetを取得します．
for(Synset s : wn.findSynsets("tree", Pos.n)){

    // 下位語を取得して表示します
    for(Synset h : s.getConnectedSynsets(Link.hypo)){

        System.out.println(h);

    }

}

wn.close();
